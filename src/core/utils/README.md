# Core Utils

Эта папка предназначена для переиспользуемых утилит, которые могут быть использованы в разных проектах.

## Назначение

- Утилиты для работы с данными
- Форматирование и парсинг
- Математические операции
- Строковые операции
- И другие переиспользуемые функции

## Правила

- Все функции должны быть чистыми (pure functions) где возможно
- Функции должны быть хорошо типизированы
- Каждая утилита должна иметь JSDoc комментарии
- Утилиты должны быть независимыми от бизнес-логики приложения
