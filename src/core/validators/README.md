# Core Validators

Эта папка предназначена для переиспользуемых валидаторов, которые могут быть использованы в разных проектах.

## Назначение

- Валидация данных
- Валидация форм
- Валидация email, URL, и других форматов
- Валидация типов данных
- И другие переиспользуемые валидаторы

## Правила

- Валидаторы должны возвращать boolean или объект с результатом валидации
- Валидаторы должны быть хорошо типизированы
- Каждый валидатор должен иметь JSDoc комментарии
- Валидаторы должны быть независимыми от бизнес-логики приложения
